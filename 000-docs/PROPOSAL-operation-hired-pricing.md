## Service Overview

Intent Solutions provides Operation Hired with a fully managed AI-powered resume generation platform that:

- Converts military documents into ATS-optimized civilian resumes
- Generates 3-document bundle (Military, Civilian, Crosswalk)
- Integrates with Polaris ATS (pending API access)
- Maintains secure document handling with configurable retention

**You focus on placements. We handle the tech.**
